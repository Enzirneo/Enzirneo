## 👨‍💻 Sobre mim

Sou desenvolvedor com foco em construção de aplicações web modernas, unindo desenvolvimento **full stack**, **arquitetura de software** e **design de interfaces**.

Tenho interesse especial em criar produtos bem organizados, escaláveis e visualmente consistentes, pensando não só no código, mas também na experiência de quem usa.

Atualmente, venho trabalhando em projetos com foco em:

- integração entre frontend, backend e regras de negócio
